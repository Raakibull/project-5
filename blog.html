Question-3: What is event delegation in the DOM?
Event Delegation is a technique where you attach a single event handler to a parent and not separate listeners to each child. This is made possible because of event bubbling, where the events bubble up from the target to the parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?
In JavaScript, you can create, add, and remove elements in the DOM (Document Object Model) using various methods.
To create a new element, you use the document.createElement() method. This creates an element.
To add an element to the DOM, you use methods like appendChild(), prepend(), or insertBefore() depending on where you want to insert the element.
To remove an element, you first need to select it, and then use the removeChild() or remove() method.
